FDC register mirroring

The diskROM is not mirrored (as far as I know), but the FDC registers can be mirrored to several places in the slot with the diskROM. The diskROM is in page 1. Here is a list of machines with disk drives or external disk drives and in which other pages the FDC registers are visible.

machine page 0 page 2 page 3
Philips VG 8230 no no no
Philips VG 8235/00 no yes no
Philips VG 8235/20 no yes no
Philips NMS 8245 no yes no
Philips NMS 8250/55 yes yes yes
Philips NMS 8280 yes yes yes
Spectravideo SVI-738 yes yes yes
Sony HB-F1XDJ no no no
Sony HBD-50 yes yes yes
Philips VY-0010 yes yes yes
Panasonic FS-FD1A yes yes yes
